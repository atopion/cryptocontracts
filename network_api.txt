API for network peers

Constructor:

peer()
Create peer-object. 


Functions:

connectToNet()
Establish connection to the p2p network

disconnectFromNet()
End connection with the network

getActivePeers()
Return a list of all currently active peers in the network

requestGraph(address)
Send a request to the peer stated in "address" to receive the graph
address (String): location as IP address from that Graph should be requested

sendGraph(address)
Send the latest local version of the graph to peer stated in "address"
address (String): location as IP address where Graph to send to
